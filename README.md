# jobpal.

jobpal. was built by me to help students with all their recruiting needs. As a fellow cooked cs grad, I understand just how many apps one can go through a day. This app will help to organize everything and give others like myself a more clear view of their path.

## Features

- Track job applications
- Track contacts reached out
- Receive reminders to check on statuses

## Getting Started

1. Clone the repository
2. Install dependencies
3. Run the application
