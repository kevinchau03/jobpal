# jobpal.

jobpal. was built by me to help students with all their recruiting needs. As a fellow cooked cs grad, I understand just how many apps one can go through a day. This app will help to organize everything and give others like myself a more clear view of their path. Let's all get that j*b

## Features

- Track job applications
- Track contacts reached out
- Receive reminders to check on statuses

## Tech Stack

JobPal is a full-stack TypeScript app built with Next.js + Express + Prisma + PostgreSQL, secured by HTTP-only cookie JWT auth, styled with Tailwind, and enhanced with modern data fetching and UI components.

## Getting Started

1. Clone the repository
2. `cd backend && npm install`
3. `cd frontend && npm install`
4. Create your own `JWT_SECRET` and `DATABASE_URL` and add to `.env`
3. Run `npm run dev`
